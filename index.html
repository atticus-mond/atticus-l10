Firebase Setup
Connect to Firebase Firestore for real-time sync across all team members — Jam, MJ, Shanne, Jorynn, Josiah, Ange, and guests. Data persists indefinitely; archive by year whenever needed.

One-time setup steps:
1. Go to console.firebase.google.com
2. Click Add project → name it atticus-l10
3. In the project: Firestore Database → Create database → Start in test mode
4. Go to Project Settings (gear icon) → Your apps → Add app → Web icon (</>)
5. Register the app, then copy the firebaseConfig object shown
6. Paste it in the field below and click Connect

Security (after testing): In Firestore Rules, change to require authentication or restrict to your domain.
Paste your firebaseConfig here — looks like:
const firebaseConfig = {
  apiKey: "AIza...",
  authDomain: "atticus-l10.firebaseapp.com",
  projectId: "atticus-l10",
  storageBucket: "atticus-l10.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123456789:web:abc123"
};
